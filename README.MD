# NA`ME 웹

개인 웹페이지입니다.

## 기술스택

- NextJS 14
  - with App Router
- StyleX
- Typescript
- ESLint, Prettier
- Storybook
- bun

## 준비

```bash
yarn install
```

## 실행

```bash
// 웹
yarn dev

// 스토리북
yarn storybook

// 테스트
yarn test

// 커버리지 생성
yarn coverage
```
